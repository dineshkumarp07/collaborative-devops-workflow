# Collaborative DevOps Workflow

## Project Overview
This project demonstrates collaborative development using GitHub DevOps workflow with feature branching and pull requests.

## Project Structure
collaborative-devops-workflow
│
├── docs
├── src
│   └── login feature files
├── README.md

## Features Implemented
- Login Feature
- Feature Branch Workflow
- Pull Request Creation
- Pull Request Merge
- Branch Deletion

## Workflow Steps Followed
1. Created repository
2. Created feature branch (feature-login)
3. Added login feature
4. Created pull request
5. Reviewed pull request
6. Merged pull request
7. Deleted feature branch
